# 📚 Gestion de Merceria -  System Marimar 

*Proyecto académico para Gestion de Merceria - Universidad Nacional de Asuncion -  Facultad Politecnica*

## 🚀 Descripción
Sistema de gestion de merceria 
Implementa funcionalidades como:
- ✔️ Registro, modificacion y eliminacion logica de Categorias
- ✔️ Registro, modificacion y eliminacion logica de Productos
- ✔️ Registro, modificacion y eliminacion logica de Compras
- ✔️ Registro, modificacion y eliminacion logica de Ventas
- ✔️ Registro, modificacion y eliminacion logica  de Stock
- ✔️ Registro de Movimientos de Stock

## 🛠️ Tecnologías
- **Backend**: Django 4.x (Python 3.x).
- **Frontend**: Bootstrap 5, HTML5, CSS3.
- **Dinámica cliente**: JavaScript (Vanilla o jQuery).
- **Base de datos**: PostgreSQL.
