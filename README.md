📦 Sistema de Inventario

Este es un Sistema de Inventario completo desarrollado para gestionar eficientemente las operaciones de un negocio. El sistema permite administrar ventas, compras, inventarios y generar reportes clave para la toma de decisiones.
🚀 Funcionalidades principales

✅ Módulo de Ventas

    Registro de ventas con detalle de productos vendidos.

    Consulta de ventas diarias, mensuales y anuales.

    Reporte de productos más vendidos.

✅ Módulo de Compras

    Registro de compras a proveedores.

    Consulta de compras diarias, mensuales y anuales.

✅ Módulo de Inventario

    Gestión de stock de productos.

    Cálculo del valor actual del inventario.

    Consulta del valor histórico del inventario.

✅ Reportes

    Valor actual del inventario.

    Valor histórico del inventario.

    Productos más vendidos.

    Ventas diarias, mensuales y anuales.

    Compras diarias, mensuales y anuales.

📊 Reportes incluidos

    Valor actual del inventario: Total del stock valorizado según el precio actual.

    Valor histórico del inventario: Seguimiento del valor a lo largo del tiempo.

    Productos más vendidos: Ranking de productos según cantidad y monto vendido.

    Ventas diarias, mensuales y anuales: Análisis de desempeño de ventas.

    Compras diarias, mensuales y anuales: Control de adquisiciones.
